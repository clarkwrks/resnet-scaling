# Markdown
Markdown syntax is used to format content within a panel. Generally, paragraph's are delineated by ending a line with two spaces followed by a return.


## Headers
Headers are used to structure content.

```markdown
# Header Level 1
## Header Level 2
### Header Level 3
#### Header Level 4
##### Header Level 5
###### Header Level 6
```
# Header Level 1
## Header Level 2
### Header Level 3
#### Header Level 4
##### Header Level 5
###### Header Level 6

## Weight and Emphasis

```markdown
**Bold Text**  
*Italicized Text*  
***Bold and Italic***  
Super<sup>script</sup>  
Sub<sub>script</sub>  
`Inline Code` 

> Block Quote  
```
